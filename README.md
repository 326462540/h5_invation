# h5_invation
这是一个h5页面，只适配了手机，电脑、pad打开样式会炸裂的。

## 页面包含：

    1、页面利用了css3动画效果，模拟信封打开效果；
    
    2、实现局部rem响应式，不像一般使用的那种全局body加font-size；
    
    3、有微信分享代码，可直接使用；
